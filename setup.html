Setup automático del Google Sheet

  1. Conectar con Google Apps Script
  2. Crear hojas y estructura
  3. Cargar datos iniciales del menú
  4. Verificar respuesta de la API
Hacé clic en el botón para configurar el spreadsheet automáticamente.
¿Qué hace este script?
Llama al endpoint ?action=setup de tu Apps Script, que:
1. Crea las 6 hojas con headers y formatos
2. Carga la carta completa con datos iniciales
3. Configura validaciones y protecciones

⚠️ Requisito previo: El Code.gs debe estar pegado y desplegado en Apps Script antes de usar este setup.
